

# Triple C Events — Website Plan

## Overview
A bold, vibrant website for **Triple C Events** — an event management company that organizes vendor malls and markets. The site will attract vendors, sponsors, and event clients while allowing a single admin to manage everything.

---

## 🏠 1. Homepage (Landing Page)
- **Hero Section** — Eye-catching banner with event tagline, next upcoming event date, and CTA buttons ("Become a Vendor", "Become a Sponsor", "Get Tickets")
- **Why Join as a Vendor?** — Benefits section (exposure, foot traffic, raffle engagement, networking, etc.)
- **Why Sponsor?** — Tiered sponsorship benefits overview (Gold, Silver, Bronze visibility levels)
- **Featured Vendors Section** — Highlighted vendors who paid for promotion, displayed prominently with their logo, product images, and social links
- **Upcoming Events** — Next events with dates, location, and ticket links
- **Past Events Gallery** — Photo gallery from previous events to build credibility
- **Raffle & Cash Prize Info** — Explain the purchase-to-raffle-ticket system
- **Contact Section** — Phone, email, social media links (Instagram: triplecccevents_, Facebook: Triple Cha-nel)

## 🛍️ 2. Vendors Page
- List of all registered vendors with:
  - Business name, description, logo
  - 1-2 product images (vertical style)
  - Social media links (Instagram, Facebook, website)
  - Contact details (phone, email, location)
- **Featured Vendors** shown at the top with special badge/highlight
- Vendor registration form for new vendors to apply

## 🤝 3. Sponsors Page
- **Tiered display**: Gold, Silver, Bronze sponsors with different visual prominence
- Sponsor logos and descriptions
- Sponsorship inquiry form for interested sponsors
- Benefits breakdown per tier

## 📅 4. Events Page
- **Upcoming Events** — Date, time, venue, description, ticket link (Eventbrite integration via link)
- **Past Events** — Gallery/recap with photos from previous events
- **"Host Your Event With Us"** CTA — Attract businesses to use Triple C Events for their event management needs

## 📝 5. Vendor Registration Form
- Business name, description, category
- Contact info (phone, email)
- Social media links
- Product image uploads (1-2 images)
- Submits inquiry to admin for review

## 📝 6. Sponsor Inquiry Form
- Company name, contact details
- Desired sponsorship tier (Gold/Silver/Bronze)
- Message/notes
- Submits inquiry to admin

## 🔧 7. Admin Panel (Password Protected)
- **Dashboard** — Overview of vendors, sponsors, events
- **Manage Vendors** — Add/edit/delete vendors, mark as "Featured", approve registrations
- **Manage Sponsors** — Add/edit/delete sponsors, assign tiers (Gold/Silver/Bronze)
- **Manage Events** — Create/edit upcoming events, add past event photos
- **Manage Featured Vendors** — Control which vendors appear in the featured section, set featured benefits description
- **Edit Homepage Content** — Update hero text, benefits sections, contact info

## 🗄️ 8. Backend (Lovable Cloud + Supabase)
- Database for vendors, sponsors, events, and admin content
- Storage for vendor product images, event photos, sponsor logos
- Single admin authentication
- User roles table for admin access control

## 🎨 9. Design Style
- **Bold & Vibrant** color scheme — energetic, event/market atmosphere
- Strong typography, bright accent colors
- Mobile-responsive design throughout
- Coffee/market event branding elements
